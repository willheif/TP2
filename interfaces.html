Les interfaces web

Chaque site web , application ou plateforme basée sur un ordinateur, est adapté pour une utilisation humaine avec la création d’une interface graphique. Par exemple le site internet Youtube qui a connu récemment une nouvelle mise à jour de son interface. Pour comblé l’arrive massive de l’idéologie associée au WEB 3 (la nouvelle version du web qui priorise les nouvelles avancées des années 2010 et de celle à venir. ) La nouvelle interface de YouTube présente une intégration de la vidéo visionnée avec l’interface interactive. Aussi une simplification des graphiques est remarquée sur la page YouTube. Plusieurs entreprises utilisent de nouveaux logiciels comme figma pour simplifier la tâche de création. Aussi l’implantation de ressource dans le site par des applications web comme les API et le web3 uikit , un open source qui vise à faciliter la conception des futurs sites. En bas à gauche, un modèle 3d crée avec l'application spline. Elle permet l'intégration intéractive de modéle 3d sur les sites web .
